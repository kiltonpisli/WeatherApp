# WeatherApp

## Live Preview
https://kiltonpisli.github.io/WeatherApp/

## Run Locally

Clone the project

```bash
  git clone https://github.com/kiltonpisli/WeatherApp.git
```

Go to the project directory

```bash
  cd ./frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Tech Stack

**FrontEnd:** ReactJS, Redux-Toolkit, RTK Query, FontAwesome, SASS

**API:** OpenWeather API
